## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/iamdevmarcos/my-trips.git
```

2. Then go to the project's folder:

```sh
cd my-trips
```

3. Install all dependencies:

```sh
yarn install
```

4. Run locally:

```sh
yarn start
```
